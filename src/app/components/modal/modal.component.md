# MODAL COMPONENT

## INSTALLATION

```
<app-modal
  [display]="manageModal.open"
>
  <div body class="dialog">
    <div dialog="header"></div>
    <div dialog="content"></div>
    <div dialog="footer"></div>
  </div>
</app-modal>
```

## PROPERTIES

- display ``boolean`` - Modal display status

## COMPONENTS

- body ``element`` - Modal content
